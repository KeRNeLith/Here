CustomResult<TError>

CustomResult<TError> is an object that represents the result/state of a treatment. It provides a custom error object in case of failure.

Either<TLeft, TRight>

Either<TLeft, TRight> is an object that represents a value of one of two possible types. Either<TLeft, TRight> is an instance of "Left" or "Right". A common use of Either<TLeft, TRight> is as an alternative to Option<T> for dealing with possible missing values. In this usage, None is replaced with a Left which can contain useful information. Right takes the place of Some(in T). Convention is that Left is used for failure and Right is used for success.

EitherLeft<TLeft>

EitherLeft<TLeft> is an object that represents an Either<TLeft, TRight> initialized in a "Left" state. Convention is that Left is used for failure.

EitherRight<TRight>

EitherRight<TRight> is an object that represents an Either<TLeft, TRight> initialized in a "Right" state. Convention is that Right is used for success.

Option<T>

Option<T> is an object that may embed something (otherwise it's nothing).

Result

Result is an object that represents the result/state of a treatment.

Result<T>

Result<T> is an object that represents the result/state of a treatment. This Result<T> embed a Value resulting of the treatment.

Result<T, TError>

Result<T, TError> is an object that represents the result/state of a treatment. This Result<T, TError> embed a Value resulting of the treatment or a custom error object in case of failed.
