Struct Either<TLeft, TRight>

Either<TLeft, TRight> is an object that represents a value of one of two possible types. Either<TLeft, TRight> is an instance of "Left" or "Right". A common use of Either<TLeft, TRight> is as an alternative to Option<T> for dealing with possible missing values. In this usage, None is replaced with a Left which can contain useful information. Right takes the place of Some(T). Convention is that Left is used for failure and Right is used for success.
